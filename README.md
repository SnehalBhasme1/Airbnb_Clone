# Airbnb Clone

## Overview

A pixel-perfect Airbnb listing page clone built with Next.js and Tailwind CSS.

## Features

- Responsive Header
- Hero Gallery
- Sticky Booking Card
- Amenities
- Reviews
- Responsive Design

## Tech Stack

- Next.js
- TypeScript
- Tailwind CSS
- Lucide React

## Installation

npm install
npm run dev

## Folder Structure

src/
components/
data/
app/

## Future Improvements

- Date picker
- Guest selector
